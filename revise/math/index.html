About this website

This is a website for the author (I'll go by T. for now) to post content related to mathematics and worldbuilding, which are my two main creative outlets. Each have projects I've been working on for a number of years, but which there hasn't been much which I've put out publicly or in places that are easy for others to find or access. My hope is that by having a website going for it it'll provide motivation for me to actually start to get stuff out into the world.

I also might post about personal stuff? There's not much to say there so I might not, though.

Right now there's not a whole lot here. I'd suggest pick out one of the navigation links along the side (or in the header), and there might be more .
